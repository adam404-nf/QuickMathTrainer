# 02 PRD

## MVP 目標
做出可用的 Flashcard 練習流程，讓使用者能快速開始、連續作答並看到即時結果。

## 核心流程
選擇練習模式 → 顯示題目 → 輸入答案 → 判斷正誤 → 顯示反饋 → 下一題 → 結束統計。

## MVP 功能
- 題目生成：先支援基礎四則、分數、小數、冪次、常見高中運算。
- 答案輸入：支援鍵盤輸入與 Enter 送出。
- 即時反饋：顯示正確／錯誤、正確答案與用時。
- 練習統計：本輪題數、正確率、平均用時、錯題列表。
- 本地記錄：用 localStorage 保存最近練習與偏好設定。

## 題目形式

### 題型比例
- 填空題：約 80%。
- 選擇題：約 20%。

### 填空題
用戶需要自行輸入答案。

例如：
- `15 × 17 = ?`
- `√144 = ?`
- `log₂(32) = ?`

### 選擇題
提供 4 個選項。

要求：
- 只有 1 個正確答案。
- 干擾項需合理。
- 避免一眼看出答案。

例如：

`25 × 16 = ?`

- A. 350
- B. 375
- C. 400
- D. 425

### 題目生成規則
- 題目數字盡量隨機。
- 盡量避免短時間內重複題目。
- 同一訓練回合內避免出現相同題目。
- 優先生成未出現過的題目。

### 使用者體驗
目標：
- 快速作答。
- 快速反饋。
- 保持流暢節奏。

避免：
- 大量重複題目。
- 題目過於固定。
- 記憶答案而非訓練計算能力。

## 邊界與擴展
- 非目標：完整大題講解、課程、商業訂閱、班級管理、複雜社群。
- 未來：章節題庫、難度分級、錯題複習、限時挑戰、進步曲線、PWA 離線使用。
