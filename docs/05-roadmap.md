# 05 Roadmap
## Phase 0：專案骨架
- 建立前端專案、基本路由、題卡 UI、樣式基礎。
- 定義 Question / Attempt / Session 型別與題目生成接口。

## Phase 1：MVP Flashcard
- 完成顯示題目 → 輸入答案 → 即時反饋 → 下一題。
- 支援基礎題型、計時、正確率、平均用時、錯題列表。
- 使用 localStorage 保存最近紀錄與設定。

## Phase 2：訓練品質
- 增加高中常見運算題型與難度分級。
- 加入錯題重練、限時模式、連續練習目標。

## Phase 3：長期演化
- 個人進步曲線、章節化題庫、PWA 離線、帳號同步、排行榜、AI 出題輔助。

## 判斷標準
每次迭代都應強化速度、準確率或數字敏感度。
