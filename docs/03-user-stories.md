# 03 User Stories

## 角色
- R1 高考備戰高中生：想在短時間內練計算速度與準確率。
- R2 數學愛好者：想用 Flashcard 做日常速算挑戰。

## MVP Stories
- US1：作為使用者，我想選擇練習模式，以便針對不同運算類型訓練。
- US2：作為使用者，我想看到單張題卡並輸入答案，以便專注完成一題。
- US3：作為使用者，我想送出後立即看到正誤，以便快速修正計算習慣。
- US4：作為使用者，我想連續進入下一題，以便形成流暢練習節奏。
- US5：作為使用者，我想在結束後看到正確率與平均用時，以便了解本輪表現。
- US6：作為使用者，我想查看錯題，以便之後針對弱點重練。

## 驗收重點
- 每題流程不可中斷；鍵盤操作要順。
- 統計必須能反映速度、準確率與錯題。
- 無登入也能完成一次完整練習。
