# Question Generators

Each question type should have its own generator.

Planned generator groups:

- Arithmetic
- Fractions
- Decimals
- Powers and roots
- Logarithms
- Trigonometry
- Combinatorics
- Probability

Each generator should:

- Receive mode, difficulty, and recent question context.
- Select from a small set of high-quality templates.
- Return a standard Question object.
- Avoid React, DOM, storage, and UI concerns.
