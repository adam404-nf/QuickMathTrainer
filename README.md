# QuickMathOwo

面向 **IAL**、**香港 DSE**、**中國國內高考** 語境的 Flashcard 式數學心算訓練平台。打開即可練：出題 → 作答 → 即時反饋 → 弱項診斷 → 專攻再練。

## 功能亮點

- **Flashcard 練習流程**：顯示題目、輸入答案、立即回饋正確與否與用時
- **多題型生成**：基礎四則、分數、冪次、多步心算；填空為主、選擇為輔
- **難度與題量**：基礎／標準／挑戰／極限；每輪 10／20／50 題
- **本輪統計**：題數、正確率、平均用時、錯題列表
- **弱項診斷**：依大題型與細項能力標籤分析偏弱處，並可啟動弱項專攻
- **本地偏好**：設定與最近練習紀錄保存在 `localStorage`，無需帳號

## 快速開始

需要 [Node.js](https://nodejs.org/)（建議 LTS）。

```bash
npm install
npm run dev
```

瀏覽器開啟終端機提示的本機網址（通常為 `http://localhost:5173`）。

## 如何使用

無需註冊帳號。開啟頁面後即可練習；難度、題量與最近紀錄會存在瀏覽器本機。

### 1. 在首頁設定練習條件

進入首頁後，先調整：

| 設定 | 選項 | 說明 |
|------|------|------|
| **難度** | 基礎／標準／挑戰／極限 | 影響題目難度，也影響弱項診斷門檻 |
| **題量** | 低檔 10 題／中檔 20 題（預設）／高檔 50 題 | 三種模式共用 |

若已練過，首頁會顯示「最近一輪」正確率，方便你對照進度。

### 2. 選擇訓練模式

#### 快速練習

適合先打穩單一題型：

1. 點選題型：**整數四則**、**分數小數** 或 **冪次根號**
2. 按「開始 …」進入該題型練習

#### 混合測驗

混合所有題型，模擬考場切換節奏。按「開始混合測驗」即可。

#### 弱項專攻

依歷史紀錄鎖定偏弱能力再練：

1. 先完成幾輪混合或快速練習（累積足夠作答紀錄）
2. 在首頁按「查看弱項」，或在結果頁按「前往專攻此項／專攻弱項」
3. 勾選要專攻的能力標籤（例如乘法、運算次序、分數）
4. 開始專攻練習

若紀錄不足（例如總題數太少），畫面會提示先多練幾輪再使用弱項專攻。

### 3. 作答與導覽

練習開始後會進入「速算進行中」：

1. **閱讀題目**（數學式以 KaTeX 渲染）
2. **作答**
   - 填空題：在輸入框鍵入答案，按「送出」或 **Enter**
   - 選擇題：點選四個選項之一後送出
3. **查看回饋**：是否正確、正確答案、本題用時
4. **下一題**：按「下一題」，或鍵盤 **→**；最後一題會變成「查看統計」
5. 可按「上一題」或鍵盤 **←** 回看已作答題目
6. 不確定時可先按「顯示答案」再繼續

中途想離開：按「返回首頁」。若本輪尚未結束，會先確認是否退出。

### 4. 看結果與再練

一輪結束後會看到本輪統計：

- 題數、正確率、平均用時
- **偏弱項目**：依大題型與細項能力標示「偏弱／穩定／資料不足」，並附簡短診斷文案
- 可選擇：**再練一輪**、**返回首頁**，或直接進入**弱項專攻**

建議節奏：**混合／快速練習 → 看弱項 → 專攻 → 再測一輪**，形成診斷閉環。

### 使用小提示

- 答案格式以題目提示為準（例如概念型根號題可能要求 `|A|` 這類固定格式）
- 設定與練習歷史存在本機；清除瀏覽器網站資料會一併清除
- 弱項判斷會同時看正確率與平均用時；樣本太少時不會強行標為偏弱

## 產品原則（摘要）

1. **Flashcard 優先** — 節奏要快，反饋要即時  
2. **輕量使用** — 打開就能練，不以帳號或社群為前提  
3. **考試語境一致** — 題目與文案對應具名升學考試，而非模糊「速算」  
4. **診斷驅動練習** — 弱項分析要能導向專攻再練
